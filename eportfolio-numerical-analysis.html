Data Activities

Ungraded: Crime Survey

Maria Ingold

Topic: Crime Survey

Using the Crime Survey for England and Wales, 2013-2014: Unrestricted Access Teaching Dataset, implement the following in R:

  • 1.2: Assess level of anti-social behaviour that the survey respondents experience in their neighbourhood by creating a summary statistic, using the antisocx variable.
  • 2.1, 2.2, 2.3: Using the bcsvictim variable, explore whether survey respondents experienced crime in previous 12 months, create a frequency table and convert into a factor variable.
  • 3.0: Create a subset of individuals who belong to 75+ age group and who were a victim of crime. Save as crime_75victim.
  • 4.1, 4.2: Create boxplot for antisocx variable, plot in yellow, outliers in red. Create a bar plot for bcsvictim variable.

Learning Outcomes

  • Outcome 1: Learning how to use R to read and manipulate data sets.
  • Outcome 2: Use R to create statistical tools like frequency table which indicates mode.
