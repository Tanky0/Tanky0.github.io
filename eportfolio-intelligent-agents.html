Development Team Project: Academic Research Online Agent

Maria Ingold

Topic

Develop an agent that is capable of Academic Research Online: finding results on a website based on search terms, extracting the data and sending to an offline location. Deliver a comprehensive design proposal report, describing how you will meet the requirements, design and implementation of the solution for the organisation.

Learning Outcomes

Learning Outcome 1

Created design proposal report for Academic Research Online agent. It used Python, requests, Beautiful Soup 4 and Pandas in collect, extract, transform. This is a client-side simple reflex agent using a DOM tree-based approach. The design is modular which works well with a PRINCE2 Agile plan and iteration an enables extension over time to include scrapy for crawling, Selenium for automation and NLTK for NLP.

Learning Outcome 2

This applied a simple reflex agent to academic research online. Research demonstrated that data changes over time and fault tolerance and error handing will be needed beyond prototype to account for volume, variety, velocity and veracity. Code will need to accommodate both representational and connectivity errors.

Learning Outcome 3

As this is an early-knowledge team, requests, Beautiful Soup and Pandas was selected for this design stage over Scrapy. Future iterations will require legislation and ethical considerations including ensuring use of robots.txt, rate limiting, not scraping at peak hours, respecting terms of use, and avoiding private data.

Learning Outcome 4

The team was fully virtual across time zones. Maria set up a shared Google Drive, with communication via WhatsApp and Microsoft Teams. Planning and iteration used a PRINCE2 Agile methodology and spreadsheet planner. Each team member created an initial outline to gain understanding and growth. Maria created a core interactive outline for thinking through design decisions and rationale, and early references and training for shared growth. This was accepted as the base outline. Maria merged and completed the first full draft which was reviewed and edited for final draft by the team. Incremental progress was shared via WhatsApp.
